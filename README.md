# Welcome to OMDb Apps

This is simple movie apps using react, redux and redux-saga middleware

![Home](/screenshot/home.png)

![Movies](/screenshot/movies.png)

How run Localy

First, Install Dependencies:

```bash
npm run install
# or
yarn install
```

Second, Run Apps

```bash
npm start
# or
yarn start
```

Test

```bash
yarn test
```

Build

```bash
yarn build
```
